隱私權政策

我們的承諾

PortfolioNow 重視您的隱私。

我們深知投資資訊的敏感性，因此採用「本地優先」的設計理念，確保您的資料安全。

資料收集

我們不收集任何個人資料。

PortfolioNow 不會收集、儲存或傳輸以下資訊：

  • 個人身份資訊（姓名、電子郵件、電話等）
  • 投資組合資料（持倉、交易記錄、損益等）
  • 裝置識別資訊
  • 使用行為數據

資料儲存

所有資料僅儲存在您的裝置與個人 iCloud 帳號中。

您輸入的所有投資資料，包括：

  • 持倉資訊
  • 交易記錄
  • 投資組合歷史快照

均使用 Apple 的 SwiftData 框架儲存在您的裝置本地儲存空間中。

iCloud 同步

PortfolioNow 使用 Apple 的 CloudKit 服務提供 iCloud 同步功能：

  • 自動備份 - 您的資料會自動同步至您的個人 iCloud 帳號
  • 多裝置同步 - 在 iPhone、iPad、Mac 等裝置間無縫同步
  • 端對端加密 - 資料傳輸與儲存均受 Apple 的安全機制保護

iCloud 同步資料僅儲存在您的個人 Apple 帳號中，我們無法存取這些資料。如需停用同步，請至 iOS 設定 → Apple ID → iCloud 關閉本 App 的 iCloud 存取權限。

網路連線

PortfolioNow 僅在以下情況連線網路：

  • 取得即時股價 - 透過第三方金融數據 API 取得股票報價，此過程不會傳送您的個人資料或投資組合資訊。

第三方服務

本 App 使用以下第三方服務取得股價資訊：

  • Yahoo Finance API - 用於取得即時股價

我們不會向這些服務傳送任何可識別您身份或投資組合內容的資訊。

資料安全

為保護您的資料安全，PortfolioNow 提供：

  • Face ID / Touch ID 鎖定 - 防止未經授權的存取
  • 金額隱藏功能 - 在公共場合保護敏感數字
  • 本地儲存 - 資料不離開您的裝置

資料匯出與備份

您可以隨時透過 App 內的匯出功能，將所有資料以 JSON 格式匯出至您選擇的位置。匯出的檔案完全由您控制。

兒童隱私

PortfolioNow 不針對 13 歲以下兒童設計，我們不會有意收集兒童的任何資訊。

政策更新

如本隱私權政策有任何變更，我們將在此頁面更新。建議您定期查閱本頁面以了解最新資訊。
